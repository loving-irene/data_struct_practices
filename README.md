### 背景说明
作为一个初学者，市面上的数据结构教程，其内容编排和讲解方式对于我来讲，显得不够简单和直接，于是
萌生了一个想法，自己写一本《数据结构练习集》，记录下自己学习数据结构的过程。

本练习集，通过提出问题、设计数据结构来解决问题这样的模式，以实战为主，在解决问题的过程中学习一
些概念。
